# QwaveCameraCV

QwaveCameraCV is a LabVIEW Camera Driver library for Raspberry Pi 3B/3B+

[**QwaveCameraCV**](https://github.com/QWaveSystems/QwaveCameraCV) is a LabVIEW Camera driver library .This package required [LINX 3.0.xx](http://sine.ni.com/nips/cds/view/p/lang/en/nid/212478) to deploy the code and run under Raspberry Pi board. The LINX library is created by Digilent and maintain the support at [labviewmakerhub.com](https://www.labviewmakerhub.com) This version of library is supported LabVIEW 2014 SP1 only and only for Non-commercial use.

Anyone can get started with the [LabVIEW 45-Day Evaluation](http://ftp.ni.com/support/softlib/labview/labview_development_system/2014%20SP1/2014sp1LV-WinEng.exe) and buy LabVIEW 2014 SP1 Home Edition to using with Raspberry Pi 3B, 3B+ and 3A+.

QwaveSys RPi Package is a peripheral VIs provides enhance functions over LINX 3.0 standard library. it's an easy to use VIs and provided many examples. It's tested with Raspberry Pi 2B/3B and some functions also works with BBB and Orange Pi (H3) boards.
